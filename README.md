# wezterm_lua

Personal WezTerm configuration repository.

## Files

- `.wezterm.lua`: main WezTerm configuration file
- `wezterm_config_links.md`: curated links to official WezTerm config docs

## Usage

1. Edit `.wezterm.lua`.
2. Reload config in WezTerm with `Ctrl+Shift+R`.

## Useful Commands

- `wezterm ls colors` - list built-in color schemes
- `wezterm show-keys` - show key bindings
- `wezterm cli` - interact with running WezTerm instances
