## Full Lua API Reference

### `wezterm` Module

Main module documentation: https://wezterm.org/config/lua/wezterm/index.html

Includes functions such as:
- `wezterm.config_builder()` - Creates a config builder
- `wezterm.action()` - Creates key actions
- `wezterm.on()` - Registers event handlers
- `wezterm.font()` - Creates font objects
- `wezterm.format()` - Formats strings
- `wezterm.glob()` - Matches files
- and more...

### Key Assignments

Complete list: https://wezterm.org/config/lua/keyassignment/index.html

Common actions:
- `ActivateTab` - Switch tabs
- `SplitPane` / `SplitHorizontal` / `SplitVertical` - Split panes
- `IncreaseFontSize` / `DecreaseFontSize` / `ResetFontSize` - Adjust font size
- `ActivateCopyMode` - Activate copy mode
- `Paste` / `Copy` - Paste/Copy
- `SendKey` / `SendString` - Send key/string
- `SpawnTab` / `SpawnWindow` - Open new tab/window
